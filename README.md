# checksum
File checksum verification tool written in Go.

## Installation
```
$ git clone https://github.com/ibnaleem/checksum.git
```
```
$ go build
```
## Usage
```
$ checksum checksum <file_path> <hash>
```

## Hashing Algorithms
This script supports the following hashing algorithms:
- MD5
- SHA1
- SHA224
- SHA256
- SHA384
- SHA512
